password-hashing
======
This fork adds a composer.json file to defuse/password-hashing

Add this to your composer.json:
```
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/thomasoberthanner/password-hashing"
        }
    ],
    "require": {
        "thomasoberthanner/password-hashing": "master"
    }
}
```

* Version 1.0
